狙击镜蒙版纹理设计指南
===================

文件名: sniper_scope.png
尺寸: 256x256 像素 (会自动缩放到屏幕大小)

设计要求:
1. 中心圆形透明区域 - 狙击镜视野
2. 周围黑色不透明区域 - 遮挡视野
3. 可选: 十字准星线条
4. 可选: 刻度标记

颜色规范:
- 透明区域: Alpha = 0 (完全透明)
- 遮挡区域: 黑色 RGB(0,0,0) Alpha = 200-255 (半透明到不透明)
- 准星线条: 白色或绿色 RGB(255,255,255) 或 RGB(0,255,0)

建议布局:
- 中心圆形直径约为纹理宽度的60-70%
- 十字准星线条宽度2-4像素
- 外围完全黑色遮挡

注意: 如果没有专门的纹理文件，系统会使用程序生成的简单蒙版
